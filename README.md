A very simple MTG Life Counter.


HTML5 App.

Dependencies
===========

node.js+npm
-----------

Install nodejs + npm and dependencies.

    npm install

bower
-----

Install bower.

    npm install -g bower

And install script dependencies.

    npm install

grunt
-----

Install grunt.

    npm install -g grunt-cli

jshint
------

Enable JS inspections .jshintrc

jscs
----

Enable JS inspections .jscsrc
